# SmartPassCloud 🚌

## Overview

SmartPassCloud is a cloud-based bus pass management system developed using React.js, Node.js, Express.js, and PostgreSQL. The system transforms traditional bus pass management into a smart digital experience by allowing users to apply online, upload photos, receive instant email notifications, track application status, and access QR-based digital passes.

## Features

* User Registration and Login
* Online Bus Pass Application
* Photo Upload Support
* Instant Email Confirmation
* QR-Based Digital Pass Generation
* Pass Status Tracking
* Download Digital Pass
* Admin Approval and Rejection System
* Cloud-Based Data Management

## Technologies Used
